## 1. network_map_2d 简介

> 一款基于 canvas、d3.js、svg等技术实现的开源的可视化网络攻击地图，使用目前最新的canvas粒子特效、同心圆特效果等。本项目为简单demo版本，支持快速移植到您的项目，可在一定程度上提高您的开发效率，快速实现基于2D canvas的网络态势攻击可视化大屏。
